## 🚀 核心能力

```
用户输入（自然语言）
    ↓
[意图识别Agent] → 解析模糊需求为结构化意图
    ↓
[知识检索Agent] → 多源知识库并行检索（6大知识源）
    ↓
[课件生成Agent] → 生成课件大纲 + 演讲稿 + 学员手册
    ↓
完整课件（10分钟）
```

## 🏗️ 多Agent架构

### 3个核心Agent

| Agent | 职责 | 核心输出 |
|------|------|---------|
| **意图识别Agent** | 解析用户自然语言需求 | 结构化JSON意图 |
| **知识检索Agent** | 多源知识库并行检索 | 筛选后的知识片段 |
| **课件生成Agent** | 生成完整培训课件 | 大纲+演讲稿+学员手册 |

### 支持的知识源

| 知识源 | 类型 | 内容特点 | 优先级 |
|--------|------|----------|--------|
| **本地文献库** | 本地文件 | 学术论文、政策文件 | 最高 |
| **本地Wiki** | 本地文件 | 151篇系统整理的概念方法 | 高 |
| **乐享知识库** | 云端/本地 | 医院内部培训资料、经验分享 | 中 |
| **IMA知识库** | 云端 | 腾讯智能体结构化内容 | 中 |
| **ONES** | 云端 | 项目管理流程文档 | 中 |
| **Obsidian** | 本地 | 个人深度思考、双向链接网络 | 低 |

## 🎬 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置知识源

编辑 `config.py`，修改以下路径：

```python
KNOWLEDGE_SOURCES = {
    'local_wiki': {
        'path': '你的Wiki路径',      # ← 修改
        'enabled': True
    },
    'obsidian': {
        'path': '你的Obsidian路径',  # ← 修改
        'enabled': True
    }
    # ...
}
```

### 3. 运行

```bash
python main.py
```

### 4. 输入需求

```
请输入培训需求：
> 给心内科主任讲45分钟低倍率病组管理，重点讲怎么识别
```

### 5. 查看输出

输出文件保存在 `output/` 目录：
- `01_intent_*.json` - 结构化意图
- `02_knowledge_*.md` - 知识片段
- `03_courseware_*.md` - 完整课件

| Part 5 | 低倍率怎么管？ | 10分钟 |

| 课件章节 | 知识来源 | 内容 |
