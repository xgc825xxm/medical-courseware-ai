# 课件截图说明

由于HTML课件需要在浏览器中打开截图，这里提供截图指南和替代展示方案。

## 截图方法

### 方法1：浏览器截图（推荐）

1. 打开课件HTML文件
   ```bash
   # 进入课件目录
   cd "E:/.opencode/科技大赛/小红书WAIC/低倍率病种管理培训课件"
   
   # 用浏览器打开
   start 低倍率病种管理-方案C.html
   ```

2. 使用浏览器截图工具
   - Chrome：F12 → Ctrl+Shift+P → 输入"capture" → 选择"Capture full size screenshot"
   - Edge：右键 → "网页捕获"

3. 截取以下页面：
   - [ ] 封面页（Slide 1）
   - [ ] 目录页（Slide 2）
   - [ ] Part 1 内容页（Slide 4）
   - [ ] Part 2 内容页（Slide 7）
   - [ ] Part 5 内容页（Slide 18）
   - [ ] 案例页（Slide 22）
   - [ ] 结尾页（Slide 29）

### 方法2：手动截图

按F11全屏，逐页截图，保存为：
- `01_封面.png`
- `02_目录.png`
- `03_Part1.png`
- `04_Part2.png`
- `05_Part5.png`
- `06_案例.png`
- `07_结尾.png`

## 建议

- 优先使用**方法1**（浏览器全页截图），效果最佳
- 截图后替换README中的图片链接
- 每张截图建议尺寸：1920×1080 或 1600×900
