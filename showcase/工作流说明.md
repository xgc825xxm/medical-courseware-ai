# 工作流展示说明

## 系统运行流程

```
┌─────────────────────────────────────────────────────────────┐
│  用户输入                                                    │
│  "给心内科主任讲45分钟低倍率病组管理，重点讲怎么识别"          │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  Step 1: 意图识别Agent (1分钟)                               │
│  ─────────────────────────────                               │
│  输入：自然语言                                               │
│  处理：Prompt模板解析                                         │
│  输出：{"topic":"低倍率病组管理", "audience":"科室主任", ...}   │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  Step 2: 知识检索Agent (2分钟)                               │
│  ─────────────────────────────                               │
│  输入：topic="低倍率病组管理"                                  │
│  处理：多源并行检索                                           │
│    ├── 本地文献库 → 2篇论文                                   │
│    ├── 本地Wiki → 5篇概念文章                                 │
│    ├── 乐享知识库 → 3篇科室经验                               │
│    ├── IMA知识库 → 2篇政策解读                                │
│    └── Obsidian → 1篇个人笔记                                 │
│  输出：5条结构化知识片段                                       │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  Step 3: 课件生成Agent (5分钟)                               │
│  ─────────────────────────────                               │
│  输入：意图 + 知识片段                                        │
│  处理：Prompt模板生成                                         │
│  输出：                                                       │
│    ├── 课件大纲（7 Part结构）                                 │
│    ├── 演讲稿（逐字稿）                                       │
│    └── 学员手册（1页）                                        │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  最终输出                                                    │
│  ├── 01_intent_低倍率病组管理.json                            │
│  ├── 02_knowledge_低倍率病组管理.md                           │
│  └── 03_courseware_低倍率病组管理.md                          │
└─────────────────────────────────────────────────────────────┘

总耗时：8分钟
```

## 效率对比

| 环节 | 传统方式 | 我们的方式 | 提升 |
|------|---------|-----------|------|
| 找资料 | 翻书+查文献+问同事，1天 | 多源知识库一键检索，5分钟 | **120倍** |
| 写大纲 | 手写+反复修改，半天 | Agent协作生成，2分钟 | **360倍** |
| 出课件 | PPT排版+配图，1天 | 标准化结构输出，即时 | **∞** |
| 写讲稿 | 自己琢磨怎么说，半天 | 逐字稿自动生成，5分钟 | **∞** |
| **总计** | **2-3天** | **8分钟** | **300倍+** |

## 录制建议

如果制作录屏/GIF：

1. **录屏内容**：
   - 终端输入命令
   - 显示3个Agent的执行过程
   - 展示最终输出文件

2. **推荐工具**：
   - ScreenToGif（Windows）
   - LICEcap（跨平台）
   - OBS Studio（专业）

3. **时长**：30-60秒

4. **尺寸**：1280×720 或 1920×1080
